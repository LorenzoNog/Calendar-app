# Calendar App 📆

Página web que replica el funcionamiento de un calendario, en donde se pueden observar tanto los días de cada mes como registrar eventos.

# ⚒️ Construída con:
- Next JS 13

# ✨ Se utiliza:
- React
- Tailwind CSS
- TypeScript
